# Gut Microbiome Diversity Analysis: Antibiotic-Associated Diarrhea (AAD)

## Project Overview
This project performs a comprehensive statistical analysis on gut microbiome diversity changes in patients treated with different antibiotic classes (FQN, OBL, PBL). The study investigates whether specific antibiotics lead to significant alterations in alpha-diversity (Shannon and Chao1 indices) and compositional shifts (Jaccard distance).

## Repository Contents
* **`Stat_Project_D.ipynb`**: A complete R-based Jupyter Notebook containing the data cleaning, descriptive statistics, and inferential analysis.
* **`stat_docs.pdf`**: Detailed documentation of the statistical methodology, hypothesis testing (T-tests, ANOVA), and linear regression models.
* **`AAD_Analysis.pptx`**: A presentation summarizing the key findings, including diversity decrease from Day 1 to Day 6 and correlation analyses.

## Skills & Tools Demonstrated
* **Languages:** R Programming.
* **Statistics:** Hypothesis testing (Welch T-test, ANOVA), Linear Modeling, and Normality testing.
* **Data Visualization:** Distribution plots, Boxplots (Day 1 vs Day 6), and Q-Q plots.
* **Concepts:** Microbiome Diversity, Clinical Data Analysis, and Statistical Inference.
